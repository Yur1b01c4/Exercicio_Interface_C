
# Sistema de Gestão de Documentos (POO)

Projeto desenvolvido durante as aulas de Programação Orientada a Objetos na FatecTaquaritinga. 
O objetivo foi aplicar conceitos de **Interfaces** e **Classes Abstratas** em C#.

## 🛠️ Tecnologias e Conceitos
- C# / .NET 8
- Programação Orientada a Objetos (Herança e Polimorfismo)
- Visual Studio 2022

## 🚀 Como rodar
1. Clone o repositório.
2. Abra o arquivo `.sln` no Visual Studio.
3. Aperte `F5` para compilar e executar.
